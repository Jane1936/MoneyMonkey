# MoneyMonkey 微信小程序

用于记录投资理财收支，支持：

- 中国 A 股
- 港股
- 美股
- 银行积存金
- 黄金/白银/铂金实物
- 虚拟币

## 功能

1. 记录每笔投资的买入时间、卖出时间、买入金额、卖出金额。
2. 自动计算每笔交易收益率。
3. 根据交易记录生成月报分析与年报分析：
   - 交易笔数
   - 买入/卖出总额
   - 已实现收益
   - 平均收益率
   - 各投资品类分布收益与平均收益率

## 目录结构

- `pages/index`：交易记录录入与列表
- `pages/report`：月报、年报分析
- `utils/report.js`：报表统计逻辑

## 使用方式

1. 用微信开发者工具打开本项目目录。
2. 在“投资记录”页新增交易。
3. 点击“查看月报 / 年报分析”查看统计结果。
