# MoneyMonkey 微信小程序

用于记录投资理财收支，支持：

- 中国 A 股、港股、美股
- 银行积存金
- 黄金/白银/铂金实物
- 虚拟币

## 本次升级

1. UI 风格升级为 iOS 风格（圆角卡片、分段选择器、浅灰背景）。
2. 投资品类为下拉选择，并新增“具体名称”输入（如股票名/币种名）。
3. 买入与卖出金额旁新增货币下拉（RMB / HKD / USD）。
4. 新增收益率曲线，支持按年、月、周、日切换查看。

## 你应该用什么 IDE？

- **微信开发者工具（必须）**：用于运行、预览、调试小程序。
- **VS Code（推荐）**：用于代码编辑和 Git 管理。

## 运行与检查效果

1. 微信开发者工具 -> 导入项目 -> 选择当前目录。
2. AppID 可使用 `touristappid` 本地调试。
3. 在“投资记录”页新增几笔数据（不同投资品类和币种）。
4. 在“分析报表”页检查：
   - 曲线是否可按年/月/周/日切换。
   - 月报/年报中的交易笔数、分币种买卖金额和收益是否符合预期。

## 目录

- `pages/index`：交易录入页
- `pages/report`：报表与收益率曲线
- `utils/report.js`：报表与曲线聚合逻辑
